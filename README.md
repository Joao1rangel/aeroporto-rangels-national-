## 📋 Sobre o Projeto
Sistema de banco de dados relacional para gerenciamento do Aeroporto Rangel's National, simulando a operação do dia 20/06/2026.

1. Abra o MySQL Workbench
2. Abra o arquivo `aeroporto_rangels_completo.sql`
3. Execute com `Ctrl + Shift + Enter`
